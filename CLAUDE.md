# stym06.github.io

Personal portfolio and blog site. Static HTML hosted on GitHub Pages.

## Tech stack
- Plain HTML + CSS (no framework)
- Font: Space Grotesk
- Theme: Dark sepia
- CSS: `/assets/css/style.css`
- Build script: `build.py` (Python, uses `.venv`)

## Writing a new blog post

1. Create a markdown file in `content/posts/<slug>.md`
2. Add frontmatter:
   ```markdown
   ---
   title: Post Title
   date: YYYY-MM-DD
   reading_time: X min read
   ---
   ```
3. Write content in markdown (supports headings, code blocks, images, links, blockquotes)
4. Run `.venv/bin/python3 build.py` to generate HTML
5. Commit and push to deploy

The build script generates `posts/<slug>/index.html` for each post and updates `posts/index.html` (the writing listing page).

## Pages
- `/` — Homepage (hand-edited: `index.html`)
- `/posts/` — Writing listing (auto-generated by build.py)
- `/posts/<slug>/` — Individual posts (auto-generated by build.py)
- `/projects/` — Projects (hand-edited: `projects/index.html`)
- `/about/` — About with resume details (hand-edited: `about/index.html`)

## Deployment
- Push to `main` branch deploys to GitHub Pages automatically
- Remote uses SSH: `git@github.com:stym06/stym06.github.io.git`

## Notes
- All internal links use trailing slashes (required for GitHub Pages)
- `.venv/` is gitignored — run `python3 -m venv .venv && .venv/bin/pip install markdown` to set up
